- Fetch:

    2) snippet
    
    3) source
    
    4) headline
    
    5) keywords
    
    6) pub_date
    
    7) byline
    
    8) page
    
    9) lead_paragraph
    
    10) multimedia - thumbnail, standard, large image
    
    11) day_of_week - facet_field

    [] Prevent reloading if no wifi detected
